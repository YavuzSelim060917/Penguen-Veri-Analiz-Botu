# Yavuz - Veri Analiz Botu v1.0

Bu proje, Python kullanarak geliştirilmiş bir masaüstü veri analiz uygulamasıdır. 
Palmer Penguins veri setini kullanarak veri temizliği yapar ve istatistiksel raporlar sunar.

## Özellikler
- **Veri Temizleme:** Eksik (NaN) değerleri otomatik olarak ayıklar.
- **Analiz:** Ortalama vücut kütlesini ve toplam temiz veri sayısını hesaplar.
- **Top List:** Veri setindeki en ağır 3 pengueni listeler.
- **Arayüz:** Tkinter kullanılarak geliştirilmiş kullanıcı dostu bir GUI.

## Kullanılan Teknolojiler
- Python 3.14+
- Pandas
- Seaborn
- Tkinter (GUI)
